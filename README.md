# chess-system-java
Chess system made with Java.

This is a didactic project. It shows many resources of Object Oriented Program as follows:

- Encapsulation
- Constructors
- Overriding
- Associations
- Encapsulation / Access Modifiers
- Enumerations
- Inheritance
- Downcasting
- Static members
- Layers pattern
- Polymorphism
- Exceptions
- Abstract method / class
